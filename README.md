# 📊 Budget vs Actual Performance Dashboard

## 📌 Project Overview

This project is an interactive Excel dashboard developed to compare Budget vs Actual performance across key financial metrics.

It helps analyze business performance using KPIs, variance analysis, Pivot Tables, Pivot Charts, and interactive slicers.

---

## 🎯 Objectives

- Compare Budget vs Actual values
- Calculate Variance and Variance %
- Analyze Profit Variance
- Track financial performance
- Support business decision-making

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Formulas
- Conditional Formatting
- Slicers
- Data Validation
